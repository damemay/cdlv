# zdlv
Experimental branch of cdlv that provides three programs: `zdlv-packer`, `zdlv-server` and modified `cdlv-menu`.

`zdlv-server` acts as a remote provider of .adv files and resources for `cdlv-menu` that are sent and received to client when needed.

`zdlv-packer` packs images into scene-contained files that are sent by `zdlv-server` to `cdlv-menu`.
